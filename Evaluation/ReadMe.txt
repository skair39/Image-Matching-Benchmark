If you use this dataset in your research publications, please cite our paper:
"Image Matching Benchmark. JiaWang Bian, Le Zhang, Yun Liu, Wen-Yan Lin, Ming-Ming Cheng and Ian D. Reid"

In the folder "Evaluation":
	1. "Evaluate.m" is a function to calculate SP and AP.
	2. "/export_fig" is used to save figures as "pdf" format. 
	3. You can use "PlotResults.m" to plot figures as presented in the paper.
	4. "PlotResults2.m" is used to plot figures of additional experiments in the paper.
